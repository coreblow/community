# CoreBlow Community

Policies and documentation for the CoreBlow community.

This repository keeps community operations separate from the product runtime:
moderation guidance, onboarding, contributor roles, and incident response.

## Documents

- `rules.md` defines baseline community conduct.
- `roles.md` defines responsibility boundaries.
- `onboarding.md` gives new contributors a first path.
- `incident-playbook.md` keeps response work calm and auditable.
